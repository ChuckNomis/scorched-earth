Detailed Description

Top-level game class — owns the ECS world and drives the main loop.

Construction spawns all initial entities (terrain, two tanks). run() ticks every system in a fixed 60 Hz loop until ESC or window close.

System order per frame:

  1. input_system
  2. ai_system
  3. physics_system
  4. collision_system
  5. damage_system
  6. lifetime_system
  7. render_system (always runs, even on game-over)
